You can now export simple HTML (*.htm) and raw XML output directly!

5- Manage References and Namespaces

You have now access to TeklaPad's brain to see what's going on. Right-Click on query area > Manage  

 

So, you can make modifications (Adding / Removing) on extra References (or Namespaces) to extend TeklaPad's capabilities!

Learn more here 

 

Hint: It's only for advanced users and can greatly affect the functionality of TeklaPad. So, change the default configurations only if you know what you're doing!

7- Custom Templates

You can now add your own template to TeklaPad's template list:

 

Hint: You must add you text files (Custom Templates) to the following folder:

[~Installation Path] / TSDB_CustomTemplates

to be displayed at the end of the template list.

 

 

8- Extended code snippets (xSnippet)

Code snippets can make writing program code quicker, easier, and more reliable. With them, you can type a few letters and then press [Tab] twice to insert a code fragment.

 

While using the first tab of TeklaPad, You can now use many extended code snippets (xSnippet) for more efficient coding!

Find more on:

User Manual > Tab-1> C# basics> Extended Code Snippets

 

Hint: You can invoke code snippet list manually with [Ctrl]+K+X (or Right-Click > Insert Snippet).
