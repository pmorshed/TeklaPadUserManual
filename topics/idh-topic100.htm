Important:  If you want getting summation of "double" type values (weight, length,...) inside query body via GetTeklaReport method, You must declare the result type explicitly in Sum() function or use generic version of this method ( GetTeklaReport<T> ):

 

let TotalWeight = g.Sum(x => (double)x.GetTeklaReport("weight net"))

 

let TotalWeight = g.Sum(x => x.GetTeklaReport<double>("weight net"))
