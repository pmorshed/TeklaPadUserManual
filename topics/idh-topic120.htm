About UI

 

Adding new notes

There are two methods for adding new notes:

 

Method-A (adding one by one):

1- Write a unique title for your new note (Required)

2- Write a content for your note (Required)

3- Press Add button:

4- Press Save button:  ->  (this action cannot be undone)

 

Method-B (batch process):

1- Drag your existing text files (*.txt or *.cs) and drop them into "Title list" area. Then all dragged items will be added automatically (file names will be considered as titles).

2- Press Save button:  ->  (this action cannot be undone)

 

Hint1: Empty text files and all files with duplicate names will be ignored (because each note must have a unique title).

Hint2: Method-B is not allowed while searching. So make sure that the search box is empty before dragging.

Hint3: In Tab-1 and Tab-2, you can easily Right-Click and choose "Add as New Note" to add your current code to saved notes:

 

After adding, Go to tab-4 and press Save button:  -> (this action cannot be undone)

 

Removing notes

1- Select a note from title list.

2- Press Remove button:

3- Press Save button:  ->  (this action cannot be undone)

 

Modifying notes

1- Select a note from title list.

2- Make any change you want (in title or content)

3- Press Modify button:

4- Press Save button:  ->  (this action cannot be undone)

 

Searching in note titles

You can use the search box to see all items that contains special word or character.

 

Hint: Clear this field to see all notes.

 

Reordering notes

1- Select a note (from title list) that you want to move.

2- Drag it upward/downward and drop it in new position of title list.

3- Press Save button:  -> (this action cannot be undone)

 

Hint: This operation is not allowed while searching. So make sure that the search box is empty before reordering.

 

Saving all notes to SavedNotes.tpad file

After changing (add/remove/modify/reorder) the note list, you must save it in order to apply changes (this action cannot be undone). After that, the "SavedNotes.tpad" file will be created (or overwritten) automatically in TeklaPad's current location and can be used for later usages.

The color of Save  button will be changed to  if any unsaved operations (add/remove/modify/reorder) detected.

 

Hint1: Don't rename/edit the "SavedNotes.tpad" file manually.

Hint2: It's highly recommended to backup your .tpad file periodically and keep it in safe place. It's the main storage for your notes.

 

Loading notes from SavedNotes.tpad file

TeklaPad always looks for this file (in its directory) while first initializing and the file will be loaded automatically.

 

Importing notes from external sources (*.tpad, *.txt , *.cs)

For loading any other external *.tpad (or *.txt, *.cs) file manually: Right-Click on "Title List" area > Import

 

Hint: You can use this method to combine multiple files

 

Hints/Tips

- Maximum capacity of note storage is 100.

- You can double-click in title list's empty space to clear the title and content fields (For creating a new note).
