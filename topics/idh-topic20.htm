Tab1: Tekla Database

This tab allows you to interact with database of Tekla Structures as simple as possible (via xAPI). The benefit of using first tab of TeklaPad (instead of second tab) is that the first tab will works in all versions of Tekla Structures without need to any additional configuration.

How?! In brief*:

 

Select any area in your model/Drawing. Your selection will be used as "Database" / "DrawingDatabase"

Write your query and "Run " it (or press F5).

Done! Now you can perform analysis (Grouping/Sorting/Searching,...) on the resultant data and export it anywhere you want.

For advanced users:

TeklaPad acts as an additional layer over original API and uses it's own extended API (xAPI), So it contains some additional features which are not available in original API of Tekla Structures. For example "GetTeklaReport" or "GetTeklaPullOut" functions are not available in original API, whereas they're accessible in first tab of TeklaPad to make workflow more efficient.
