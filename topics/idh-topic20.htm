Tab1: Tekla Database

This tab allows you to interact with Tekla database as simple as possible.

How?! In brief:

 

Select any area in your model. Your selection will be used as "Database"

Write your query and "Run" it (or press F5).

Done! Now you can perform analysis (Group/Sort/Search) on the resultant data and export it anywhere you want.

 

 

 

For advanced users:

TeklaPad acts as an additional layer over original API. So it contains some additional features which are not available in original API of Tekla Structures.

For example "GetTeklaReport" or "GetTeklaPullOut" functions are not available in original API, whereas they're accessible in first tab of TeklaPad to make workflow more efficient.
