User Manual

 Version 3.0.6

 

TeklaPad is a smart API playground for Tekla Structures. It's an unofficial C# gateway that allows users to interact with Tekla database via API power to running queries (getting data from Tekla model) or commands (sending data to Tekla model). Interacting with Tekla Structures through OpenAPI was never simpler than this!

 

Key features:

- Get/Send data from/to Tekla Structures with API power

- Extended API (xAPI) & Extended code snippets (xSnippet)

- Data mining on DXF, IFC, XML, CSV,..

- No need to Visual Studio

- Online library

- Limitless & Free!

 

 

 

 

 

 

 

 

Don't worry if you are not familiar with C# or API, this tool will help you practice and increase your knowledge. It's not just an extension, It's the new way of thinking... (FAQ)

 

 

 

TeklaPad contains independent tabs:

 

 

Tekla Database

ØThe main Tab of TeklaPad.

ØAll settings will be done automatically!

ØCompatible with all versions of Tekla Structures (no need to any configuration)

ØSupports xAPI (Extended API) , xSnippet, Templates,...

 

C# Console

ØTiny C# compiler.

ØAll settings must be done manually (coding from the scratch)

 

Library

ØOnline library that contains some examples for using the first or second tab of TeklaPad.

Saved notes

ØBuilt-in note storage that allows you to save your notes (queries/commands) for later usages.
