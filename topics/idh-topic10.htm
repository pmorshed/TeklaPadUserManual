User Manual

 Version 3.0.0

 

TeklaPad is an unofficial C# gateway that allows users to interact with Tekla database via API & Extended API (xAPI) power to running queries (getting data from model) or commands (sending data to model).

It has also a tiny compiler for creating console applications or plug-ins for using in Tekla Structures or any other reason.

 

 

Key features:

- Interact with Tekla Database

- Get/Send data from/to Tekla Structures with API power

- Practice API and learn C#

- Create extension or plug-in

- No need to Visual Studio

- Limitless & Free!

 

 

 

TeklaPad is the first API playground for Tekla Structures. One tool with endless capabilities! Interacting with Tekla Structures through OpenAPI was never simpler than this!

 

Don't worry if you are not familiar with C# or API, this tool will help you practice and increase your knowledge. It's not just an extension, It's the new way of thinking... (FAQ)

 

TeklaPad contains four independent tabs. Which one are you interested in? 

 

 

 

 

 

Tekla Database

ØThe main Tab of TeklaPad.

ØAll settings will be done automatically!

ØCompatible with all versions of Tekla Structures (no need to any configuration)

ØSupports xAPI (Extended API) , xSnippet, Templates,...

 

C# Console

ØTiny C# compiler.

ØAll settings must be done manually (coding from the scratch)

 

Library

ØOnline library that contains some examples for using the first or second tab of TeklaPad.

Saved notes

ØBuilt-in note storage that allows you to save your notes (queries/commands) for later usages.
