User Manual

 Version 2.0.0

 

TeklaPad is an unofficial C# gateway that allows users to interact with Tekla database via API & xAPI power to running queries or commands with LINQ or standard syntax (learn more).

It has also a tiny compiler for creating console applications or plug-ins for using in Tekla Structures or any other reason.

 

 

Key features:

- Interacting with Tekla Database

- Get data from Tekla Structures with API power

- Practice API and learn C#

- Create extension or plug-in

- No need to Visual Studio

- Free!

 

 

 

TeklaPad is the first API assistant for Tekla Structures. one tool for unlimited tasks! It also has an exclusive xAPI to simplify communicating with Tekla Database. Interacting with Tekla Structures (Through OpenAPI) was never simpler than this!

 

Don't worry if you are not familiar with C# or API, this tool will help you practice and increase your knowledge. It's not just an extension, It's the new way of thinking... (FAQ)
