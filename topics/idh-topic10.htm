 User Manual

 Dev. by P.Morshed

 

 

TeklaPad is an unofficial C# gateway for interacting with Tekla

database. It has also a tiny compiler for creating any console

application or plug-in for using in Tekla or any other reason.

 

 

Key features:

- Interacting with Tekla Database

- Get any data/report with API power

- Practice API and learn C#

- Create extension or plug-in

- No need to Visual Studio
