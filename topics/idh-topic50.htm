Sample-11:

Command: Creating backup for preliminary marks in "user field-1" from all selected parts

Note1: Make sure that you have selected an area in your model (as Database) with Part selection mode:

Note2: Use GetTeklaUDA function to get user defined attribute text. its input is flexible and is not case sensitive (by default).

Note3: Keep in mind that the input of SetUserProperty Method is not flexible and is always case sensitive (Because it belongs to the original Tekla API  not xAPI ).

Note4: You can also use TeklaPad's safe converters (AsDouble , AsInteger, AsDate) to get numeric or DateTime values from UDA text fields.

Note5: While using the first tab of TeklaPad, You must determine a result for showing in data table. So, If you don't want to pass anything to data table, just end your command with "return null;"
