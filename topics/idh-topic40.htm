>Connection status

After running TeklaPad, It will start searching for Tekla Structures model. If model be found and everything goes fine, the first tab will be activated (Fig.2), otherwise this tab remains inactive (Fig.1).

 

Fig.1

 

 

Fig.2

 

 

 

 

 

Important Notes:

1- You can retry to establish a connection by clicking on "Refresh"  button.

2- If you faced a screen, as shown in Fig.1, make sure that the Tekla Structure's model is open and click "Refresh".

3- TeklaPad can only connect to single Tekla model. So the connection will be lost if multiple models are running at once.

4- If you find any unsolved/unexpected issues, send us your session log . Find more here 

 

>UI Buttons

 

Run

Run query over Database/DrawingDatabase (current selection in Model/Drawing). Click here to find out more about C# syntax and query concept.

Refresh

Refresh the connection between Tekla Structures model and TeklaPad

Export the data

Export data to Excel or just copy it to the clipboard. "Export to Excel" feature is available in version 1.0.3 and later versions.

 

Note1: If you choose "Export to Excel" option, then you must also specify the location for output file. The output file name contains date and time.

Note2: If you choose "Copy to clipboard" option, then you can paste it (Ctrl+v) in any text editor program (like Notepad or...).

Clear table

Remove all data from data table

Find/Replace

Open "Find/Replace" dialog box (or press Ctrl+F in query area)

Comment

Comment out the selected lines. Click here to find out more about C# syntax.

Uncomment

Uncomment the selected lines. Click here to find out more about C# syntax.

Remove bookmarks

Remove all bookmarks

Help

Online or offline documentation of software.

Note: It is recommended to use online manual to get the latest updates. Click on help button while holding the Control key to get online manual.

>Working with data table:

Data table can perform group, sort, search and subtotal over the data.

 

1- Grouping:

Drag column's header into "group area" to group by that column. You can also create nested groups:

[Data table view]

2- Sorting:

Click on each column's header to sort in ascending or descending order. You can also do right click on column's header to choose ordering options.

3- Searching:

Click on data table (to make sure that the table is active) and then press "Ctrl+F"

4- Get total and subtotal:

Click on column's header while holding the Control key to get total or subtotal
